A ready-to-use Streamlit app for:
- Uploading multiple CSV tables
- Exploring data interactively
- Chatting with an LLM contextualized by your tables
- Generating insights and SDG project plans

## Installation

```bash
pip install -r requirements.txt
```

Create a `.env` file:

```bash
OPENAI_API_KEY=your_key_here
```

## Run the app

```bash
streamlit run app.py
```

## Features

* Multi-table upload (CSV)
* LLM chat augmented with your data (markdown previews)
* Data explorer & basic analytics
* Priority scoring tool
* Automatic SDG project plan generator
